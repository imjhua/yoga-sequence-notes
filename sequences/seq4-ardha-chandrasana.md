# 빈야사-아르다찬드라사나

**포커스:** 균형 · 옆구리 열기 · **피크포즈:** 아르다찬드라사나

## 개요

- **핵심 cue**: 워밍업(수카 → 사상가) / 다운독·테이블 워밍업 / 태양경배 / 빌드업 시퀀스 → 아르다찬드라 피크

## 수업 메모

- 사상가사나에서 롤업까지 한 호흡으로 이어가기
- 테이블: 한쪽씩 측면 열기 강조
- 빌드업 2회: 1회는 우티타 트리코나(측면) → 2회는 아르다찬드라(피크)
- 호흡과 함께 동작 전환
