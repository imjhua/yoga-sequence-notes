# 하타-우르드바다누라사나

**포커스:** 백밴드 · 가슴 열기 · 역굴곡 · **피크포즈:** 우르드바다누라사나

## 개요

- **핵심 cue**: 팔꿈치 넓게 · 가슴 하늘로 · 머리 편안함

## 수업 메모

- 역굴곡 시퀀스로 흉추 가동성과 어깨 안정성 단계적 강화
- 바즈라사나에서 호흡으로 팔꿈치 위치 다시 설정
- 비라사나는 가슴 앞면·옆면 통합 스트레치

| 포즈 | # | 동작 |
|---|---|---|

| | 2 | `inhale` 뒤통수에 대고 팔꿈치 좌우로 넓게 |
| | 3 | `exhale` 모으기 턱 쇄골 가까이 |
