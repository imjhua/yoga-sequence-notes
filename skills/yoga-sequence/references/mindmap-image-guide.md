# Mind Map Image Guide

마인드맵은 **세로 흐름형** SVG로 만든다. 모바일(핸드폰)에서 수업 중 한눈에 보는 용도.

## 저장 위치

```
public/mindmaps/seq{N}-mindmap.svg
```

## 레이아웃 (권장)

```
┌─────────────────────┐  보라 — 목표자세
│ 사마코나사나        │
│ 목표자세 · cue      │
└──────────┬──────────┘
           │ 연결선
┌──────────▼──────────┐  초록 — 빌드업 단계
│ 포즈명              │
│ 이 단계의 목적/cue  │
└──────────┬──────────┘
           │
          ...
┌──────────▼──────────┐  주황 — 피크
│ 사마코나사나        │
│ 피크 cue            │
└─────────────────────┘
```

- **viewBox 너비 380px** — 모바일 한 열
- 각 박스: **포즈명(14px)** + **부제(11px)**
- 색 (어스 톤): 배경 `#F5F2ED`, 목표 `#5A5A40`, 단계 `#FFFCF9`, 피크 `#EBE8E2`

## MD embed

```markdown
## 마인드맵

![시퀀스 마인드맵](/mindmaps/seq1-mindmap.svg)
```

## 생성 방법

한글 SVG는 **Python/shell로 UTF-8 저장** (에디터 직접 저장 시 인코딩 깨질 수 있음).

```bash
python3 scripts/generate-mindmap.py seq1
```

또는 Miro/XMind export → PNG → `public/mindmaps/` (SVG 권장).

## 단계 수

- **8~10개** 이내로 압축 (수업 흐름 요약)
- 상세 포즈는 MD 표에, 마인드맵은 **큰 그림만**
